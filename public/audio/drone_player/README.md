# Drone Player Audio Files

Audio files for the Drone Player should be placed in this directory structure:

## Directory Structure

```
public/audio/drone_player/
├── tanpura/
│   ├── A.mp3
│   ├── Asharp.mp3 (or A#.mp3)
│   ├── B.mp3
│   ├── C.mp3
│   ├── Csharp.mp3 (or C#.mp3)
│   ├── D.mp3
│   ├── Dsharp.mp3 (or D#.mp3)
│   ├── E.mp3
│   ├── F.mp3
│   ├── Fsharp.mp3 (or F#.mp3)
│   ├── G.mp3
│   └── Gsharp.mp3 (or G#.mp3)
└── shrutibox/
    ├── A.mp3
    ├── Asharp.mp3
    ├── B.mp3
    ├── C.mp3
    ├── Csharp.mp3
    ├── D.mp3
    ├── Dsharp.mp3
    ├── E.mp3
    ├── F.mp3
    ├── Fsharp.mp3
    ├── G.mp3
    └── Gsharp.mp3
```

## Note Mapping

- C, C#, D, D#, E, F, F#, G, G#, A, A#, B
- Flats are mapped to sharps (Bb → Asharp, Eb → Dsharp, etc.)
- Files should be loopable MP3s for continuous drone playback
